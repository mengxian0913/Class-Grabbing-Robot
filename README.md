## 如何使用

```sh
git clone git@github.com:mengxian0913/Class-Grabbing-Robot.git
pip3 install PIL
pip3 install pytesseract
pip3 install selenium
```

> 到 main/config.py 設定帳號密碼

```py
ACCOUNT = "Account"
PASSWORD = "Password"
```

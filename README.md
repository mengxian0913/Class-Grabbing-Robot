## 如何使用

```sh
git clone git@github.com:mengxian0913/Class-Grabbing-Robot.git
pip install PIL
pip install pytesseract
pip install selenium
```

> 到 main/config.py 設定帳號密碼

```py
ACCOUNT = "Account"
PASSWORD = "Password"
```

> 啟動

```sh
python main/app.py
```
